# ⌨️ Atividade 2: Interagindo com a Tela

**Objetivo:** Fazer o robô buscar um texto na Wikipédia.

## 🛠️ O que você deve fazer:
1. Abra o arquivo `test_busca.py`.
2. Você precisará importar a classe `By` (já está no arquivo) para encontrar a barra de pesquisa da Wikipédia.
3. Use o comando `.send_keys()` para digitar "Teste de software" e o comando `.submit()` para dar Enter.
4. Valide se a próxima página carregou a pesquisa correta!
