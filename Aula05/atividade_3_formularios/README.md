# 🛒 Atividade 3: Preenchendo Formulários (Login)

**Objetivo:** Automatizar o login em um e-commerce.

## 📖 O Cenário
Usaremos o **SauceDemo** (`https://www.saucedemo.com/`), um site criado especificamente para a comunidade mundial de QAs treinar automação.

## 🛠️ O que você deve fazer:
1. Abra `test_login.py`.
2. O seu robô deve digitar o usuário `standard_user` e a senha `secret_sauce`.
3. Depois, deve clicar no botão de Login.
4. Para validar se o login deu certo, verifique se a URL atual mudou para a página de inventário.
